Este arquivo é apenas um lembrete do que foi feito para ativar o ambiente em outro servidor

1. Criei uma pasta para colocar os arquivos do projeto

2. Copiei todo conteúdo para a pasta do projeto

3. Criei o virtualenv.
    - No VS Code, usei abri a paleta de comandos com CTRL+SHIFT+P e usei a opção Python: Create Environment
    - No console, o comando é "pip install -r requirements.txt"

4. O VS Code encontrou o arquivo requirements.txt e instalou as dependências necessárias

5. Utilizei o comando "python manage.py runserver" para subir o servidor.
